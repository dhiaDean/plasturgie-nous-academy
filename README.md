# Plasturgie-Nous Academy

## Overview
Plasturgie-Nous Academy is an inter-company training platform designed to support skill development in the plastics manufacturing sector through structured theoretical and practical learning.

## System Architecture
- Backend: Spring Boot, Oracle Database
- Web Application: React
- Admin Dashboard: React
- Mobile Application: Expo React Native

## Project Structure
- backend/ – REST API and business logic
- web/ – Learner-facing web platform
- admin/ – Administration dashboard
- mobile/ – Mobile application
